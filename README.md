## AI Code Reviewer
An intelligent full-stack web application that leverages Large Language Models (LLMs) to perform automated code reviews across multiple programming languages. The platform analyzes source code for bugs, security vulnerabilities, performance issues, and coding best practices, while providing detailed feedback, review history tracking, PDF export, and a modern dark-mode user interface.
### 🚀 Features
* AI-powered code analysis and review
* Multi-language support (Python, Java, JavaScript, C++, Go, TypeScript)
* Review history management
* Export review reports as PDF
* Dark/Light mode support
* User authentication
* Dockerized deployment
* Responsive and modern UI
### 🛠️ Tech Stack
**Frontend**

**Backend**

**Database**

**DevOps & Deployment**

### 📌 Project Goal
To streamline the code review process by providing developers with instant AI-generated feedback on code quality, security, performance, and maintainability, improving development efficiency and software quality.
